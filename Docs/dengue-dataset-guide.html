01
Target Objects
  • Plastic drum PRIORITY
  • Metal drum / barrel PRIORITY
  • Discarded tire PRIORITY
  • Flower pot / saucer HIGH
  • Tarpaulin pool HIGH
  • Uncovered container HIGH
  • Bucket HIGH
  • Styrofoam container HIGH
  • Bamboo stump HIGH
  • Coconut shell MED
  • Clogged gutter / drain MED
  • Cemetery vase MED
Any detected object = potential breeding site — water state is not annotated.

Shoot wet, dry, empty, full — all states are valid detections.
03
Distance Guide
  • Close (1–1.5m) — object fills 60–80% frame · shows water, texture, material · 20% of shots
  • Medium (2–4m) — object fills 30–50% frame · natural context visible · 50% of shots ← MAIN
  • Far (5–10m) — object fills 10–25% frame · multiple objects per frame · 30% of shots
04
Angle Guide
  • Eye-level (0°) — drums, tires, barrels, any tall container
  • Diagonal (45°) ★ — most shots, most natural human perspective
  • Top-down (90°) — buckets, pots, saucers, coconut shells, anything open-top
05
Lighting Conditions
  • Overcast / cloudy — even light, no harsh shadow · shoot baseline here first
  • Bright sunlight — common in PH · must be represented in dataset
  • Under shade / tree — mosquitoes prefer shade · critical context
06
Shoot Locations
  • Residential barangay — primary real-world deployment context
  • Vacant lots / construction sites — tires, blocks, tarps · high density
  • Cemetery — flower vases · extremely productive breeding sites
  • Market / palengke backyard — drums, open containers, waste
  • TIP Manila campus — accessible · easy permission
07
Never Do This
  • Same angle + distance for every shot
  • Only isolated objects on clean backgrounds
  • Always center the object perfectly in frame
  • Same location every session — model will overfit
  • Zoom — move your body closer instead
  • Flash, filters, portrait mode, or HDR
08
5-Shot Checklist Per Object
  • Shot 1 — Medium · eye-level · natural
  • Shot 2 — Medium · 45° diagonal
  • Shot 3 — Close · top-down · show inside
  • Shot 4 — Far · show full scene + surroundings
  • Shot 5 — Repeat Shot 1 under different light
